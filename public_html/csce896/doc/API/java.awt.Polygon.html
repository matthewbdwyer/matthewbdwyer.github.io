public boolean inside(int  x, int  y)




Determines if the specified point is inside this polygon. This method uses 
an even-odd insideness rule (also known as an "alternating rule") to determine whether the pont  is inside this polygon.




It is based on code by Hanpeter van Vliet (hvvliet@inter.nl.net).





Parameters:

x
- the xcoordinate of the point to be tested


y
- the y coordinate of the point to be tested 



Returns:

true if the point  is inside this polygon; false otherwise.
