Survey Data

This directory contains the raw survey data as of August, 1998. The data is organized into a collection of five files:

  • ALL.raw: contains all of the collected property specifications.
  • US.raw: contains property specifications written by the developers of the property patterns system and their colleagues (and students).
  • THEM.raw: contains collected property specifications from sources other than US (see above).
  • BEFORE.raw: contains property specifications written by US before the patterns system was available.
  • AFTER.raw: contains property specifications written by US after the patterns system was available.

Many of the collected specifications were extracted from the literature. Citations for the papers from which specifications are drawn are given in specs.bib.
